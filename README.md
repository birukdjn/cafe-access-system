# Cafe Access System

A minimal NFC-based student cafe access scanner built with Next.js.

## Features

- **NFC Scanning Only**: Simple, focused contactless student ID card scanning
- **Real-time Processing**: Instant access verification with immediate feedback
- **Secure & Hygienic**: Contactless technology for safe student identification

## Requirements

- Modern browser with Web NFC API support (Chrome/Edge on Android)
- NFC-enabled device
- Node.js 20+ installed

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your NFC-enabled device.

## Usage

1. Click "Start NFC Scanning"
2. Hold student NFC cards near your device
3. View instant access results

## Technology Stack

- Next.js 16 with App Router
- React 19
- TypeScript
- Tailwind CSS
- Web NFC API
- Axios for API calls
